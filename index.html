プライバシーポリシー

このプライバシーポリシーでは、あなたの個人情報をどのように取り扱うかについて説明します。

ユーザのトラッキングに使用されるデータ

次のデータは、他社のアプリやWebサイト間でユーザをトラッキングする目的で使用される場合があります：

  • 連絡先情報
  • メールアドレス
  • 電話番号
  • 所在地
  • 名前
  • 位置情報
  • 詳細な位置情報
  • ユーザコンテンツ
  • メールまたはテキストメッセージ
  • 写真またはビデオ
  • ID
  • ユーザID
  • デバイスID
  • 診断
  • クラッシュデータ
  • パフォーマンスデータ

ユーザに関連付けられたデータ

次のデータは収集され、ユーザの個人情報に関連付けられる場合があります：

  • 連絡先情報
  • メールアドレス
  • 電話番号
  • 所在地
  • 名前
  • 位置情報
  • 詳細な位置情報
  • ユーザコンテンツ
  • メールまたはテキストメッセージ
  • 写真またはビデオ
  • ID
  • ユーザID
  • デバイスID
  • 診断
  • クラッシュデータ
  • パフォーマンスデータ

情報の使用方法

収集した情報は以下の目的で使用します：

  • サービスの提供と改善
  • カスタマーサポートの提供
  • アプリの利用状況の分析
  • パーソナライズされた体験の提供
  • マーケティングおよびリマーケティング活動の実施

情報の共有

私たちは以下の目的で第三者と情報を共有することがあります：

  • サービス提供のためのサードパーティサービスプロバイダとの連携
  • 法律や規制に基づく要求への対応

情報の保護

私たちは以下のセキュリティ対策を講じて情報を保護します：

  • データ暗号化
  • アクセス制限

ユーザーの権利

ユーザーは以下の権利を有します：

  • 自分の個人情報へのアクセス、修正、削除の要求
  • データ処理に対する異議の申し立て

プライバシーポリシーの変更

私たちはプライバシーポリシーを随時変更することがあります。変更がある場合は、アプリ内またはウェブサイト上で通知します。
